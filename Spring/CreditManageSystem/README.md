# CreditManageSystem

## 프로젝트 소개
IT정보공학과 졸업 학점 관리 시스템

## 기능
|기능|기능 상세|
|:---:|---|
|이수 과목 관리|과목 검색<br>과목 추가<br>과목 삭제<br>이수 상태 저장|
|졸업 시뮬레이션|과목 검색<br>과목 추가<br>과목 삭제<br>졸업 시뮬레이션 결과 보기|
|회원가입 및 로그인|회원가입<br>로그인|

### 사용 기술
- ReactJS
- Spring
- Oracle
